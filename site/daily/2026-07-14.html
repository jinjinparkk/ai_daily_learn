오늘의 기초 · Fundamentals

손실 함수 Loss Function
이게 뭐야? · What is it
ENA loss function is a formula that measures how wrong a model's prediction is compared to the correct answer.
KO손실 함수는 모델의 예측이 정답과 비교해 얼마나 틀렸는지를 재는 수식입니다.
ENIt turns the abstract idea of 'error' into a single number that the model tries to make as small as possible.
KO이는 '오차'라는 추상적 개념을 모델이 최대한 작게 만들려고 하는 하나의 숫자로 바꿔 줍니다.
ENWithout a loss function, a model would have no clear signal telling it whether it is improving.
KO손실 함수가 없다면 모델은 자신이 나아지고 있는지 알려 주는 명확한 신호를 갖지 못합니다.
작동 원리 · How it works
ENFirst, the model makes a prediction for a given input.
KO먼저 모델은 주어진 입력에 대해 예측을 합니다.
ENThe loss function then compares this prediction with the true label and outputs a loss value.
KO그다음 손실 함수는 이 예측을 실제 정답과 비교하여 손실 값을 출력합니다.
ENA high loss means the prediction is far off, while a loss near zero means it is very accurate.
KO손실이 크면 예측이 많이 빗나간 것이고, 손실이 0에 가까우면 매우 정확한 것입니다.
ENDuring training, the model uses the gradient of the loss to adjust its parameters in the direction that reduces error.
KO학습 중에 모델은 손실의 기울기를 이용해 오차를 줄이는 방향으로 파라미터를 조정합니다.
ENCommon choices include mean squared error for regression and cross-entropy for classification.
KO흔히 회귀에는 평균 제곱 오차를, 분류에는 교차 엔트로피를 사용합니다.
ENChoosing the right loss function is important because it defines exactly what the model is optimizing for.
KO올바른 손실 함수 선택은 모델이 정확히 무엇을 최적화할지 정의하기 때문에 중요합니다.
🧩 비유로 이해 · Analogy
ENThink of a loss function like a golf score: the fewer strokes, the better, and each swing you adjust based on how far you missed the hole.
KO손실 함수를 골프 점수처럼 생각해 보세요. 타수가 적을수록 좋고, 홀에서 얼마나 벗어났는지에 따라 매번 스윙을 조정하는 것과 같습니다.
기본 용어 · Key terms
Loss (손실)예측과 정답 사이의 오차를 나타내는 하나의 수치.
Mean Squared Error (평균 제곱 오차)예측과 실제 값의 차이를 제곱해 평균낸 회귀용 손실 함수.
Cross-Entropy (교차 엔트로피)분류 문제에서 예측 확률 분포와 정답 분포의 차이를 재는 손실 함수.
Gradient (기울기)손실을 줄이기 위해 파라미터를 어느 방향으로 바꿔야 하는지 알려 주는 미분 값.
Optimization (최적화)손실을 최소화하도록 모델 파라미터를 반복적으로 조정하는 과정.

오늘의 핵심 개념 · Key Concept

시각적 사전학습 Visual Pretraining
ENMost foundation models are trained on plain text extracted from documents and web pages.
KO대부분의 파운데이션 모델은 문서와 웹페이지에서 추출한 순수 텍스트로 학습됩니다.
ENBut figures, equations, and page layouts carry information that text extraction throws away.
KO하지만 그림, 수식, 페이지 레이아웃에는 텍스트 추출 과정에서 버려지는 정보가 담겨 있습니다.
ENVisual pretraining feeds the model the visual form of documents directly, without converting them to text.
KO시각적 사전학습은 문서를 텍스트로 변환하지 않고 그 시각적 형태 그대로 모델에 입력합니다.
ENThe idea is that learning from how information looks can build stronger language intelligence.
KO핵심 아이디어는 정보가 어떻게 보이는지로부터 학습하면 더 강력한 언어 지능을 구축할 수 있다는 것입니다.
ENRecent experiments show this approach can outperform text-only pretraining on the same source material.
KO최근 실험은 이 방식이 동일한 원본 자료에서 텍스트만 사용한 사전학습을 능가할 수 있음을 보여 줍니다.
왜 지금? · Why now
ENAs models run out of clean text, tapping into visually rich documents opens a new, scalable source of knowledge.
KO모델이 사용할 깨끗한 텍스트가 고갈되는 상황에서, 시각적으로 풍부한 문서를 활용하는 것은 확장 가능한 새로운 지식 원천을 열어 줍니다.
