오늘의 기초 · Fundamentals

셀프 어텐션 Self-Attention
이게 뭐야? · What is it
ENSelf-attention is a mechanism that lets each element in a sequence look at every other element to decide what is relevant.
KO셀프 어텐션은 시퀀스의 각 요소가 다른 모든 요소를 살펴보며 무엇이 중요한지 판단하게 해주는 메커니즘입니다.
ENIt is the core engine behind the Transformer architecture used in most large language models today.
KO이것은 오늘날 대부분의 대형 언어 모델에 쓰이는 트랜스포머 구조의 핵심 엔진입니다.
ENUnlike older methods that read text strictly left to right, self-attention can connect distant words directly.
KO텍스트를 엄격히 왼쪽에서 오른쪽으로만 읽던 기존 방식과 달리, 셀프 어텐션은 멀리 떨어진 단어끼리도 직접 연결할 수 있습니다.
작동 원리 · How it works
ENEach token is turned into three vectors: a query, a key, and a value.
KO각 토큰은 쿼리, 키, 밸류라는 세 개의 벡터로 변환됩니다.
ENThe query of one token is compared with the keys of all tokens to measure how related they are.
KO한 토큰의 쿼리를 모든 토큰의 키와 비교하여 서로 얼마나 관련 있는지를 측정합니다.
ENThese comparison scores are converted into weights using a softmax, so they add up to one.
KO이 비교 점수들은 소프트맥스를 통해 가중치로 변환되며, 합이 1이 되도록 만들어집니다.
ENEach token then builds a new representation by taking a weighted sum of all the value vectors.
KO그러면 각 토큰은 모든 밸류 벡터의 가중합을 취해 새로운 표현을 만듭니다.
ENModels usually run several attention 'heads' in parallel, each learning different types of relationships.
KO모델은 보통 여러 개의 어텐션 '헤드'를 병렬로 실행하며, 각 헤드는 서로 다른 종류의 관계를 학습합니다.
ENA mask can be added so that a token only attends to earlier tokens, which is important for text generation.
KO마스크를 추가하면 토큰이 이전 토큰만 참조하게 만들 수 있는데, 이는 텍스트 생성에 중요합니다.
🧩 비유로 이해 · Analogy
ENThink of a group discussion where everyone asks a question (query) and listens to everyone else's name tag (key); you pay more attention to the people whose expertise (value) matches your question.
KO모두가 질문(쿼리)을 던지고 서로의 이름표(키)를 살피는 그룹 토론을 떠올려 보세요. 당신은 자신의 질문에 맞는 전문성(밸류)을 가진 사람에게 더 집중하게 됩니다.
기본 용어 · Key terms
Query/Key/Value어텐션 계산에 쓰이는 세 벡터로, 각각 '무엇을 찾는가', '무엇을 가졌는가', '실제 전달할 정보'를 나타냄
Attention weight한 토큰이 다른 토큰에 얼마나 집중할지를 나타내는 값, 소프트맥스로 정규화됨
Multi-head attention여러 어텐션을 병렬로 수행해 다양한 관계를 동시에 학습하는 방식
Causal mask미래 토큰을 보지 못하게 막아 순차적 생성이 가능하게 하는 장치
Token모델이 처리하는 최소 단위로, 단어나 단어 조각에 해당

오늘의 핵심 개념 · Key Concept

밀집 패치 특징 Dense Patch Features
ENA Vision Transformer splits an image into small patches and creates a feature vector for each one.
KO비전 트랜스포머는 이미지를 작은 패치들로 나누고 각 패치마다 특징 벡터를 만듭니다.
ENThese many per-patch vectors are called dense features because they keep fine spatial detail.
KO이렇게 만들어진 여러 패치별 벡터를 밀집 특징이라고 부르며, 세밀한 공간 정보를 유지합니다.
ENMany robot systems compress an image into a single global token, which throws away useful detail.
KO많은 로봇 시스템은 이미지를 하나의 전역 토큰으로 압축하는데, 이 과정에서 유용한 세부 정보가 사라집니다.
ENUsing dense patch tokens directly lets a policy see both what and where objects are.
KO밀집 패치 토큰을 직접 사용하면 정책 모델이 물체가 무엇인지와 어디에 있는지를 함께 볼 수 있습니다.
ENThe challenge is doing this efficiently without paying the cost of a huge vision-language model.
KO관건은 거대한 비전-언어 모델의 비용을 치르지 않고 이를 효율적으로 해내는 것입니다.
왜 지금? · Why now
ENToday's Patch Policy paper shows how attention over dense patches can make robot control both accurate and fast.
KO오늘 소개하는 Patch Policy 논문은 밀집 패치에 대한 어텐션이 로봇 제어를 정확하면서도 빠르게 만들 수 있음을 보여줍니다.

핵심 영어 표현 · Vocabulary

self-attention
셀프 어텐션; 시퀀스 내 각 요소가 다른 요소들과의 관련성을 계산하는 메커니즘
“Self-attention allows the model to weigh the importance of every word in a sentence.”
셀프 어텐션은 모델이 문장 속 모든 단어의 중요도를 저울질할 수 있게 합니다.
dense features
밀집 특징; 이미지의 각 패치마다 세밀하게 만들어진 특징 벡터들
“The robot policy consumes dense features to understand the exact position of objects.”
이 로봇 정책은 물체의 정확한 위치를 파악하기 위해 밀집 특징을 사용합니다.
fine-tune
미세조정하다; 사전학습된 모델을 특정 목적에 맞게 추가 학습시키다
“They fine-tune a vision-language model to produce a new similarity metric.”
그들은 새로운 유사성 지표를 만들기 위해 비전-언어 모델을 미세조정합니다.
context pruning
맥락 가지치기; 불필요한 입력 내용을 잘라내어 연산을 줄이는 기법
“Context pruning helps coding agents save tokens without losing accuracy.”
맥락 가지치기는 코딩 에이전트가 정확도를 잃지 않으면서 토큰을 절약하도록 돕습니다.
distill
증류하다; 큰 교사 모델의 지식을 작은 학생 모델로 옮기다
“The compact encoder is distilled from a billion-parameter teacher model.”
그 작은 인코더는 10억 파라미터 교사 모델에서 증류되었습니다.
out-of-distribution
분포 밖의; 학습 데이터와 다른 새로운 데이터를 가리키는 표현
“The metric generalizes reliably to out-of-distribution images.”
그 지표는 분포 밖의 이미지에도 안정적으로 일반화됩니다.
causal mask
인과 마스크; 토큰이 미래 정보를 보지 못하게 막는 장치
