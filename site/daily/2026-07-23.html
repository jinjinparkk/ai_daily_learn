오늘의 기초 · Fundamentals

트랜스포머 구조 Transformer Architecture
이게 뭐야? · What is it
ENThe Transformer is a neural network design introduced in 2017 that powers almost every modern large language model.
KO트랜스포머는 2017년에 소개된 신경망 설계로, 거의 모든 현대 대형 언어 모델의 기반이 됩니다.
ENIts key idea is 'attention', a mechanism that lets the model weigh how relevant every word is to every other word.
KO핵심 아이디어는 '어텐션'으로, 모든 단어가 다른 모든 단어와 얼마나 관련 있는지를 모델이 가늠하게 해주는 메커니즘입니다.
ENUnlike older models that read text strictly left to right, the Transformer can look at an entire sequence at once.
KO텍스트를 엄격하게 왼쪽에서 오른쪽으로 읽던 이전 모델과 달리, 트랜스포머는 전체 시퀀스를 한 번에 볼 수 있습니다.
작동 원리 · How it works
ENFirst, each word or token is turned into a vector of numbers called an embedding.
KO먼저 각 단어나 토큰은 임베딩이라 불리는 숫자 벡터로 변환됩니다.
ENBecause attention has no built-in sense of order, positional encodings are added so the model knows word positions.
KO어텐션 자체에는 순서 개념이 없기 때문에, 모델이 단어 위치를 알 수 있도록 위치 인코딩이 더해집니다.
ENIn the self-attention step, each token creates a query, a key, and a value, and compares its query with every key.
KO셀프 어텐션 단계에서 각 토큰은 쿼리, 키, 값을 만들고, 자신의 쿼리를 모든 키와 비교합니다.
ENThese comparisons produce attention scores that decide how much each token should focus on the others.
KO이 비교는 어텐션 점수를 만들어, 각 토큰이 다른 토큰에 얼마나 집중할지를 결정합니다.
ENThe results pass through a feed-forward network, and this whole block is stacked many times to build depth.
KO그 결과는 피드포워드 신경망을 통과하며, 이 블록 전체가 여러 번 쌓여 깊이를 만듭니다.
ENTechniques like residual connections and layer normalization keep training stable across these many layers.
KO잔차 연결과 레이어 정규화 같은 기법은 이 많은 층에 걸쳐 학습을 안정적으로 유지해줍니다.
🧩 비유로 이해 · Analogy
ENThink of a group meeting where, before speaking, each person quickly scans everyone else to decide whose points matter most to them right now.
KO회의에서 말하기 전에 각 사람이 지금 자신에게 누구의 발언이 가장 중요한지 재빨리 살펴보는 상황을 떠올려 보세요.
기본 용어 · Key terms
Attention각 토큰이 다른 토큰들과 얼마나 관련 있는지 가중치를 매기는 메커니즘.
Embedding단어나 토큰을 의미를 담은 숫자 벡터로 표현한 것.
Positional Encoding순서 정보가 없는 어텐션에 단어의 위치 정보를 더해주는 방법.
Feed-Forward Network어텐션 결과를 각 위치별로 추가 변환하는 완전 연결 신경망.
Layer Normalization각 층의 값 분포를 안정화해 학습을 원활하게 만드는 정규화 기법.

What is the core mechanism that makes the Transformer architecture work?

Attention alone treats tokens as an unordered set, so positional encodings add information about their order.
