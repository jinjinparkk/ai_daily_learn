오늘의 기초 · Fundamentals

활성화 함수 Activation Function
이게 뭐야? · What is it
ENAn activation function is a small mathematical operation applied to the output of a neuron in a neural network.
KO활성화 함수는 신경망 뉴런의 출력에 적용되는 작은 수학적 연산입니다.
ENIts main job is to decide how strongly a neuron should 'fire' and to introduce nonlinearity into the model.
KO그 주된 역할은 뉴런이 얼마나 강하게 '발화'할지 결정하고 모델에 비선형성을 도입하는 것입니다.
ENWithout it, stacking many layers would still behave like a single linear layer.
KO활성화 함수가 없으면 여러 층을 쌓아도 결국 하나의 선형 층처럼 동작할 뿐입니다.
작동 원리 · How it works
ENEach neuron first computes a weighted sum of its inputs plus a bias.
KO각 뉴런은 먼저 입력들의 가중합에 편향(bias)을 더해 계산합니다.
ENThe activation function then transforms this raw number into the neuron's final output.
KO그다음 활성화 함수가 이 원시 수치를 뉴런의 최종 출력으로 변환합니다.
ENReLU, the most common choice, keeps positive values unchanged and turns negative values into zero.
KO가장 흔히 쓰이는 ReLU는 양수는 그대로 두고 음수는 0으로 바꿉니다.
ENSigmoid squeezes any number into a range between 0 and 1, which is useful for probabilities.
KO시그모이드는 어떤 수든 0과 1 사이로 압축하는데, 이는 확률을 표현할 때 유용합니다.
ENBecause these functions are nonlinear, the network can learn curved, complex decision boundaries.
KO이 함수들은 비선형이기 때문에 신경망은 곡선적이고 복잡한 결정 경계를 학습할 수 있습니다.
ENDuring training, the function must be differentiable so gradients can flow backward and update the weights.
KO학습 중에는 기울기가 역방향으로 흘러 가중치를 갱신할 수 있도록 함수가 미분 가능해야 합니다.
🧩 비유로 이해 · Analogy
ENThink of an activation function like a dimmer switch on a light: it does not just flip on or off, but smoothly controls how much signal passes through based on the input.
KO활성화 함수는 조명의 디머 스위치와 같다고 생각하세요. 단순히 켜고 끄는 것이 아니라, 입력에 따라 얼마나 많은 신호를 통과시킬지 부드럽게 조절합니다.
기본 용어 · Key terms
Nonlinearity (비선형성)입력과 출력이 단순 비례하지 않는 성질로, 복잡한 패턴 학습을 가능하게 함.
ReLURectified Linear Unit. 음수를 0으로 만들고 양수는 그대로 두는 대표적 활성화 함수.
Sigmoid (시그모이드)값을 0과 1 사이로 압축하는 S자 모양의 함수.
Vanishing gradient (기울기 소실)깊은 신경망에서 기울기가 너무 작아져 학습이 잘 안 되는 현상.
Differentiable (미분 가능)기울기를 계산할 수 있는 성질로, 역전파 학습에 필요함.

보상 모델 Reward Model
ENA reward model is a system that scores how good a model's output is, guiding reinforcement learning.
KO보상 모델은 모델 출력이 얼마나 좋은지를 점수 매겨 강화학습을 안내하는 시스템입니다.
ENTraditionally, building one requires collecting human preference labels and training a separate model.
KO전통적으로 이를 만들려면 사람의 선호 라벨을 수집하고 별도의 모델을 학습해야 합니다.
ENToday's SpectraReward paper shows a training-free alternative: reuse a pretrained multimodal model as the judge.
KO오늘의 SpectraReward 논문은 학습이 필요 없는 대안을 제시합니다. 사전학습된 멀티모달 모델을 그대로 심판으로 재활용하는 것입니다.
ENIt measures how well the original text prompt can be recovered from the generated image.
KO이 방법은 생성된 이미지로부터 원래 텍스트 프롬프트를 얼마나 잘 복원할 수 있는지를 측정합니다.
ENA higher recovery likelihood means the image matches the prompt well, so it earns a higher reward.
KO복원 확률이 높을수록 이미지가 프롬프트와 잘 맞는다는 뜻이므로 더 높은 보상을 받습니다.
왜 지금? · Why now
ENAs AI enters the agentic era, teams want to improve models without the heavy cost of new labels or extra reward models.
KOAI가 에이전트 시대로 접어들면서, 팀들은 새로운 라벨이나 추가 보상 모델의 무거운 비용 없이 모델을 개선하려 합니다.

Q. '불공정한 심판의 내부' 논문은 LLM 심판 편향에 대해 무엇을 발견했나요?

The paper shows bias corresponds to a specific low-dimensional subspace in hidden states, which can be steered both ways.
